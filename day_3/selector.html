selectors

selector is used to select the element in the html

selector is used to apply css to the selected element

selector is used to js to the selected elements

Types of selectors

  1. Element selector
  2. id selector
  3. class selector
  4. group selector

Element selector

element selector is used to slelect the elements based on tag name

class selector

  • class selector isused to select the element based on class name
  • class selector no need to unique
  • we can class selector with .class name for css style

id selector

  • is selector is used to select element based on id name
  • id selector must be unique
  • we can use is selector with #id name for css style

Rules :

  1. class/id name shoud have alphabets and numbers (-,_)
  2. class/id name not start with the number
  3. classname and the idname can be same
